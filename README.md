# 🤖 AI Chatbot using Groq API

## 📌 Project Description
This is an AI-powered chatbot web application that leverages Groq's ultra-fast LLMs to deliver intelligent and real-time conversational experiences. Built using Python and integrated with the Groq API, the chatbot can understand natural language queries and provide context-aware, meaningful responses.

---

## 🚀 Features

- 💬 **AI Chat Interface**  
  Interact with the chatbot using natural language and receive smart, real-time replies.

- ⚡ **Groq API Integration**  
  Utilizes Groq’s LPU-powered backend for ultra-fast and low-latency responses.

- 🌐 **Web Interface**  
  Clean, responsive, and user-friendly web application for seamless interaction.

---

## 🛠️ Tech Stack
- Python  
- Groq API  
- Django 
- HTML, CSS, JavaScript  

---

## 📷 How It Works
1. User enters a query in the chat interface  
2. Request is sent to Groq API  
3. Model processes the query  
4. Response is displayed instantly  

---
